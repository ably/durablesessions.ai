# Durable Sessions

**Durable Sessions** is the emerging infrastructure layer between AI agents and the humans they serve. It addresses a gap in the current AI stack: sessions that survive disconnects, span devices, and outlive any single connection.

AI agents have state. Durable execution platforms persist workflow state. Databases store data. But no layer cleanly owns the *session* — the stateful, real-time interaction between an agent and a human across every boundary: disconnects, devices, modalities, and time.

Durable Sessions is that layer.
